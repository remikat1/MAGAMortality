# Local build steps

## Prereqs

- Node.js **20+** (recommended: install via `nvm`, `fnm`, or `asdf`)

## Run locally

```bash
cd app
npm install
npm run dev
```

## Production build (optional local check)

```bash
cd app
npm run build
npm run preview
```

## Edit data

Update `app/public/data/dataset.json`, then reload the dev server.
